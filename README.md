 Project Overview
Portable medical kit for continuous monitoring of vital signs (heart rate, 
SpO2, body temperature) with emergency alerting system for elderly patients 
and chronic disease management.

 Features
✅ Heart rate monitoring (30-200 bpm)
✅ Blood oxygen saturation (SpO2 70-100%)
✅ Contactless body temperature (±0.5°C accuracy)
✅ Real-time LCD display
✅ Cloud data logging (Firebase)
✅ Emergency SMS alerts
✅ LED status indicators

 Hardware Components
- NodeMCU ESP8266
- MAX30102 Pulse Oximeter & Heart Rate Sensor
- MLX90614 Contactless IR Thermometer
- 16x2 LCD Display with I2C
- SIM800L GSM Module
- LEDs, Buzzer, etc.

 Setup Instructions
1. Install Arduino IDE
2. Install required libraries (see docs/setup.md)
3. Upload code to NodeMCU
4. Configure Firebase credentials
5. Test sensors individually
6. Integrate and calibrate
